# 專注草原 (Pomodoro-Timer)
## APP 介紹
一款簡約的番茄鐘 APP，讓你在令人心曠神怡的專注草原上，每次前進一小步，並在蟲鳴鳥叫聲中小憩片刻。

## 練習目標
- 倒數計時
- 進度條控制
- 音檔播放
